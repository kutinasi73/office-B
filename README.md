行政書士（office-B）

## 概要
中小建設会社の経営者・現場社長をターゲットとした、「親しみやすさと相談しやすさ」に特化した特定技能採用支援LPです。
行政書士特有の「堅苦しさ」を排除し、現場の目線に立った共感設計を行うことで、心理的ハードルを下げ、新規問い合わせ獲得を最大化させることを目的としています。

## 制作意図（課題解決アプローチ）
1. 「現場社長」のインサイトを突いた共感設計
- ターゲット: 40代〜60代の中小建設会社経営者。多忙で手続きの煩雑さに頭を抱えている現場主義の社長。
- 戦略: 難しい法律用語を避け、現場で実際に起こっている人手不足の苦悩を言語化。あえて親しみやすいトーンを採用することで、「士業への相談＝難しい・高い」という先入観を払拭し、相談のハードルを最小化しました。

2. 成果から逆算した離脱防止レイアウト
- 構成: ヒートマップ分析を想定し、ユーザーが疑問や不安を感じるポイントに先回りして「理由」「実績」「Q&A」を配置。
- UI: スマートフォンでの閲覧がメインとなるため、現場の合間でも片手で操作・確認ができるよう、情報の優先順位を整理したモバイルファースト設計を徹底しました。

3. 視認性を重視したCTA戦略
- 解決策: スクロール量に関わらず、ユーザーが「相談したい」と思った瞬間にアクションへ移れるよう、視認性の高いCTA（予約・問い合わせ）ボタンを戦略的に配置しました。

## 工夫した点
- 軽量・高速なユーザー体験: 現場の不安定な通信環境でもストレスなく閲覧できるよう、Vanilla JSによる軽量な実装と画像最適化を行い、表示速度を追求しました。
- セマンティックなマークアップ: 構造化を意識したHTML記述により、SEO（検索エンジン最適化）とアクセシビリティの基盤を構築。
- WordPressではプラグインを追加しフォームを設定しました。

## 学んだこと・今後の課題
- 学んだこと:
同一サービスであっても、ターゲットの属性（堅実派か親しみやすさ重視か）によってデザインの「正解」が大きく異なることの重要性。

- 今後の改善点:
- 手続きの簡便さをより視覚的に伝えるインフォグラフィックの導入。
- フォーム入力の負担を減らすEFO（入力フォーム最適化）のさらなる強化。
